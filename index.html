madmaan

Madmaan is a dynamically and strong typed language that simplifies JavaScript's syntax and looks mad, MAN. Hate semi-colons? Well, try out exclamation marks! Does using "!" for "not" not make sense? Okay, use "NOT" instead, MAN. Plus, we know how ugly { } can be. SO. NO. CURLY. BRACKETS, MAN. This language explodes(compiles) into JavaScript, now go get MAD, MAAN.

About

madmaan is a programming language born from the minds of 7 students who are in a constant search to acquire more street cred. Mitchell, Alex, Daniel Mondo, Arash, Anthony and Nick, based madmaan off of Javascript, striving to make it easier to read and perhaps a little bit more "fun" to code. Also, if you look closely, the first letters of our names combine to make the name of the programming language.
